CSS Nedir?

Cascading Style Sheet: web sayfalarının görünümünü ve düzenini tasarlamak için kullanılan bir stil dilidir. Html ile yapılandırılmış bir web sayfasının görsel stilini tanımlamak için kullanılır. Css web tasarımında esneklik ve kontrol sağlayarak içerik ile tasarımın birbirinden ayrılmasını sağlar.



Css'in Temel Özellikleri

  • Stil Tanımlama
  • Yerleşim Düzeni
  • Mobil Uyum
  • Yeniden Kullanılabilirlik

Html bir web sayfasını biçimlendirmek için oluşturulmamıştır. Sadece içeriğini tanımlamak için oluşturulmuştur.
